The ArchivesSpace release candidate v4.2.0-RC1 is now available for early testing at [https://github.com/archivesspace/archivesspace/releases/tag/v4.2.0-RC1](https://github.com/archivesspace/archivesspace/releases/tag/v4.2.0-RC1). It can be downloaded from GitHub or explored on a live testing server at [http://rebranch.archivesspace.org/staff](http://rebranch.archivesspace.org/staff) using standard admin credentials. This pre-release provides a chance to validate changes and share feedback ahead of the official launch.

Please send all feedback to <a href="mailto:ArchivesSpaceHome@lyrasis.org">ArchivesSpaceHome@lyrasis.org</a> by March 27.
